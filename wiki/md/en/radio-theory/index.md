# Radio Theory — MeshCore

This section covers how MeshCore radio communication works: from the LoRa physical layer and packet protocol to routing, service mechanisms, monitoring, and network engineering.

- **Foundations** — radio model, node roles, comparison with LoRaWAN
- **Radio Theory** — frequency, power, link budget, propagation, antennas
- **LoRa PHY** — modulation, LoRa frame, airtime, radio profile
- **MeshCore Packets** — packet format, addressing, encryption, payloads
- **Routing** — flood, direct routing, path hash, regions, channel access
- **Service Mechanisms** — advert, ACK, trace
- **Monitoring** — RSSI, SNR, interference, statistics
- **Network Engineering** — design, capacity, security, migration
